# react-sac
Search area component for React

## Development

Install dependencies

`yarn`

Start a development server

`yarn start`

### Build package

Inside directory `./react-sac/`

	> yarn build

## NPM Link

Inside directory `./react-sac/`

	> npm link
	> npm ../app/node_modules/react`

Inside directory `./app/`

	> npm link react-sac

### Unlink

Inside directory `./react-sac/`

	> npm unlink

Inside directory `./app/`

	> npm unlink react-sac
